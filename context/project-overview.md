# Hotel Management System PRD (Project Overview)

## Project Title
Hotel Management System (HMS)

## Tech Stack
- PHP
- Tailwind CSS
- MySQL
- HTML5
- JavaScript

## Project Vision
Develop a secure, modern web-based Hotel Management System for small and medium-sized hotels. The system centralizes reservations, front desk, room management, housekeeping, billing, guest management, finance, reporting and administration.

The Restaurant Management System remains a separate application and will integrate with the Hotel Management System for services such as room charging.

## Objectives
- Digitize hotel operations.
- Improve guest experience.
- Simplify reservations and check-in/check-out.
- Track room occupancy.
- Automate billing and reporting.
- Support modular expansion.
